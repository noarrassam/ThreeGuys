# Web Services Project

## Description
It's a web services project that let users to add a feedback about specific car, so users can add a review, or rating for a specific car.

## Functionalities

* Persistence.

* JSP's and Servlets.

* JAXWS.

* Amazon DB Instance.

* WSDL.

## Features
* Manage User Info.
